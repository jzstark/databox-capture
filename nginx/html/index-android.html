HTTP Proxy

  1. Connect to WIFI network.
  2. Go to Settings --> WIFI
  3. Long tap on connected network's name.
  4. Modify network config, check the Show advanced options box.
  5. Choose Manual from the Proxy Setings drop-down menu.
  6. Enter 178.62.33.140 as Proxy host name. Enter 3128 as Proxy port

SOCKS5 Proxy

Follow Step 1-5 as in HTTP Proxy. Then choose Manual from the Proxy Setings drop-down menu. Enter 178.62.33.140 as Proxy host name. Enter 1194 as Proxy port. Click Save.

L2TP VPN

  1. Configure and activate the VPN: Launch the Settings, in the Wireless & Networks section. Tap VPN. (You may need to setup PIN code protection first.) Tap the + icon in the top-right of the screen.
  2. Enter required information as below:
    • Name: Databox VPN
    • Type: L2TP/IPSec PSK
    • Server addres: 178.62.33.140
    • IPSec pre-shared key : 654321
  3. Tap Save. Tap the Databox VPN connection:
    • Username: databox
    • Password: 123456
  4. Check the Save account information checkbox. Tap Connect. Once connected, you will see a VPN icon in the notification bar.

You can verify that your traffic is being routed properly by looking up your IP address. It should say Your public IP address is 178.62.33.140.
